FLEX

La idea principal detrás del diseño flexible es darle al contenedor la capacidad de alterar el ancho / alto de sus artículos (y el orden) para llenar mejor el espacio disponible (principalmente para acomodar todo tipo de dispositivos y tamaños de pantalla). Un contenedor flexible expande los elementos para llenar el espacio libre disponible o los reduce para evitar el desbordamiento.

USO

Para decirle a un contenedor que sus items se comportaran en FLEX necesitamos implementar el siguiente código:


.container {
  display: flex; /* or inline-flex */
}

flex-direction

Esta propiedad establece el eje principal, definiendo así la dirección en la que se colocan los elementos en el contenedor . Ej:


.container {
   display: flex; /* or inline-flex */
   flex-direction: row
}

flex-grow

Esto define la capacidad de un elemento flexible para crecer si es necesario. Acepta un valor en unidades que sirve como proporción. Ej:


.item {
  flex-grow: 10; /* default 0 */
}

flex-wrap

Por defecto, todos los elementos flexibles intentarán encajar en una línea. Puede cambiar eso y permitir que los elementos se ajusten según sea necesario con esta propiedad.


.container{
  flex-wrap: wrap;
}

align-items

Esto define el comportamiento predeterminado de cómo se distribuyen los elementos flexibles a lo largo del eje transversal en la línea actual.


.container {
  align-items: flex-start;
}

justify-content

Esto define la alineación a lo largo del eje principal. Ayuda a distribuir las sobras de espacio libre extra cuando todos los elementos han alcanzado su tamaño máximo.


.container {
  justify-content:  space-around;
}

GRID

CSS Grid-Layout es el sistema de diseño más potente disponible en CSS. Es un sistema bidimensional, lo que significa que puede manejar tanto columnas como filas, a diferencia de flexbox, que es en gran medida un sistema unidimensional. Trabaja con el Diseño de cuadrícula aplicando reglas CSS tanto a un elemento primario como a los elementos secundarios o hijos de ese elemento.
